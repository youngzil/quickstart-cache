# RedisJSON


## Redis对JSON的支持RedisJSON

RedisJSON 是一个Redis模块，在 Redis 中提供 JSON 支持。RedisJSON 允许您在 Redis 中存储、更新和检索 JSON 值，就像使用任何其他 Redis 数据类型一样。RedisJSON 还可以与RediSearch无缝协作，让您可以索引和查询 JSON 文档。
